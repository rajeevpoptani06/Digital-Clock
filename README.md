# Digital-Clock
A premium Digital Clock built with HTML, CSS and JavaScript.


## Features
- Live digital clock with smooth animations
- Switch between 12-hour and 24-hour format
- Dynamic greeting based on time of day
- Built-in Pomodoro Focus Timer
- Battery and network status display
- Dark modern design
- Smooth animations


## How to Run
1. Download all the files
2. Open the folder
3. Double click on index.html
4. Clock will open in your browser


## Built With
- HTML
- CSS
- JavaScript
